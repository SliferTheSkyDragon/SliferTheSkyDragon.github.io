Does ;release duplicates release my Shinies, Legendaries, or Goldens?

+

No. It only releases your duplicate Commons, Uncommons, Rares, and Super Rares.
The only way to release Legendary pokemons is to do it Manually
Golden cannot be released.

Can you prevent the release of Pokemon That you want to keep?

+

You can use these command to Lock some Pokemons you dont want to Release
;release lock {pokename or dexnumber} -> locks a Pokemon from being released.
;release lock {rarity} -> lock a specific rarity type from being released.
You can also unlock them using ;release unlock {pokename or dexnumber}
